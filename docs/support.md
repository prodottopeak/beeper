 # Beeper Support

  Thank you for using Beeper.

  If you need help, have a question, or want to report a bug, contact:

  prodottopeak@gmail.com

  ## Common Topics

  ### Notifications and alarms
  Make sure notification permissions are enabled in iPhone or iPad Settings for Beeper.

  ### Premium purchase restore
  If you already purchased Beeper Premium, open the premium screen in the app and use Restore Purchases.

  ### App feedback
  If something is not working as expected, please include your device model, iOS version, and a short description of the issue.
